Tiered Root Module Manager

SukiTier implements a sophisticated tiered mounting system with cascading validation. Modules are organized into four strict tiers with mandatory integrity checks and dependency chains, ensuring system stability even when modifying the kernel or core system.

Fail-Safe & Rollback (Rescue Sentry)

SukiTier includes an automated Rescue Sentry script (`rescue_sentry.sh`). If verification fails at any tier during boot, the system automatically reverts to the last known good snapshot.

OTA Patching Strategy

SukiTier survives updates by detecting the Inactive Slot (A/B partitions). The `OTAPatching.kt` service patches Tier 1 modules onto the inactive slot in the background, ensuring root persists after reboot.
